Linguistics Dual Task<br><br>
This code features a dual task for use in linguistics research. The experiment features a dot pattern working memory task and a scalar implicature sentence judgement task based on De Neys & Schaeken (2007), Dieussaert et al. (2011) and Marty & Chemla (2013).<br>
This code is intended to work in a Qualtrics environment.
